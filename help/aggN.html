aggNR Documentation

aggN aggregate vector of single age (year) data to N-year groups

Description

This only makes sense if N is 5 or 10 (or 2 I guess), since it uses modulo to find groups Ages are assumed to start at 0 and count up in single ages.

Usage

  aggN(x, N)

Arguments

x

the vector of single-age-classified data

N

desired width of interval, e.g., 5 or 10
